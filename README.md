# Fire and ice

Quick project using [An API of Ice And Fire](https://anapioficeandfire.com/)
This project was bootstrapped with [Create React App](https://github.com/facebook/create-react-app).

## Getting started

```bash
$> yarn
$> yarn start
```

## Stack

- React
- TypeScript
- styled-components
